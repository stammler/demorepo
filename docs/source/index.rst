Welcome to Calculators's documentation!
=======================================

**Calculator** is a simple program for basic arithmetic operations.

Check out the :doc:`installation` section for further information how to install
the Calculator.

.. note::

   This project is meant to be a minimum working example of the usage of Sphinx,
   how to host the documentation on Read the Docs. It is furthermore an example
   for unit test.

   This project has been created for the Code Coffee seminar at the University
   Observatory of the Ludwig Maximilian University of Munich.
